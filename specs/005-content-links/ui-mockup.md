## 8. リンクボタンの詳細デザイン

```
ボタン構造:
┌─────────────────────────────────┐
│ [icon] [label] [ExternalLink]   │
└─────────────────────────────────┘
  ↑      ↑        ↑
  Emoji  テキスト  外部リンク
         ラベル    アイコン

例:
┌─────────────────────────────────┐
│ 🌐 公式サイト 🔗                │
└─────────────────────────────────┘

スタイル:
- variant="outline"
- size="sm"
- className="flex items-center gap-2"
- hover: bg-sky-50, border-sky-300
```

## まとめ

このUIモックアップは、実装後のユーザー体験を視覚的に示しています。

**特徴**:
- 🎨 **シンプルで明確**: Emojiアイコンで直感的に理解できる
- 📱 **レスポンシブ**: モバイルとデスクトップで適切に表示
- 🔒 **セキュア**: 新しいタブで安全に外部サイトを開く
- ♿ **アクセシブル**: スクリーンリーダーに対応
- 🚀 **高速**: 追加のライブラリ不要で軽量

---

**参考**: これらのモックアップは設計段階のイメージです。実装時に調整が入る可能性があります。
